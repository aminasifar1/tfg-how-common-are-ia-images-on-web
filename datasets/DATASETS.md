# Datasets

The image datasets generated by this project are hosted on Hugging Face:

**https://huggingface.co/datasets/aminasifar1/tfg-web-images**

## Files

| File | Description | Source |
|------|-------------|--------|
| `scrape_results_websites.tar.gz` | Images scraped from 25 websites across 5 sectors (news, arts, education, e-commerce, tourism) | `scraping/batch_scraper.py` |
| `wayback_results_news.tar.gz` | Historical images from 5 news websites via Wayback Machine (2020-2025) | `scraping/wayback_machine.py` |

## Structure

### scrape_results_websites.tar.gz

```
sites/
  001_lavanguardia-com__la-vanguardia/
    images/            # Downloaded images
    metadata/
      images_metadata.csv   # Per-image metadata (url, tag, size, zone, hashes)
  002_rtve-es__rtve-noticias/
    ...
batch_summary.json
```

### wayback_results_news.tar.gz

```
sites/
  bbc-co-uk/
    2020/   2021/   2022/   2023/   2024/   2025/
    manifest.csv
  elmundo-es/
    ...
wayback_summary.json
```
